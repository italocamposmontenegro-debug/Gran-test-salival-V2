# Gran Test Integrado de Control Salival (GTCS)

Aplicación web **offline-friendly** y **100% local** (sin servidor), para uso clínico interno de Fonoaudiólogos/as en evaluación de sialorrea.

## Instalación / Uso
### Opción A — GitHub Pages
1. Subir estos archivos a la raíz del repositorio.
2. En GitHub: **Settings → Pages → Deploy from a branch → main → /(root)**.
3. Abrir la URL de Pages.

### Opción B — Local
Abrir `index.html` en un navegador moderno.

## Datos y privacidad
- Los datos se guardan en `localStorage` del navegador.
- La app **no envía datos a internet**.
- Para eliminar todo: botón **Borrar datos**.

## Exportaciones
- Copiar informe (texto).
- Exportar JSON.
- Exportar CSV.
- Exportar PDF (impresión del navegador).

## Nota
Esta herramienta apoya el registro clínico interno; **no reemplaza el juicio profesional**.
